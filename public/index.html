Spotify MCP Server

Model Context Protocol server for the Spotify Web API

A Model Context Protocol (MCP) server providing full coverage of the Spotify Web API — 80 tools across 13 categories including playback control, library management, playlists, search, and more.

Quick Setup

Claude.ai

  1. Go to Customize > Connectors > Add Custom Connector
  2. Set Name to Spotify and URL to https://sptfy-mcp.online/mcp
  3. Save and authorize with your Spotify account

Claude Desktop / Claude Code / Cursor

Add to your MCP client config:

{
  "mcpServers": {
    "spotify": {
      "type": "streamable-http",
      "url": "https://sptfy-mcp.online/mcp"
    }
  }
}
